# ASSPR

Даний проєкт є десктопним додатком на базі **C# Windows Forms**, розробленим для автоматизації складних матричних обчислень та розв'язання задач лінійного програмування (ЗЛП).

## 🌟 Основні можливості

Проєкт охоплює три ключові розділи обчислювальної математики:

### **Частина А: Матрична алгебра та СЛАР**
* **Пошук оберненої матриці** $A^{-1}$ за допомогою процедури звичайних жорданових виключень (ЗЖВ).
* **Обчислення рангу** довільної прямокутної матриці.
* **Розв'язання систем лінійних алгебраїчних рівнянь (СЛАР)** трьома методами:
    * Побудова та використання оберненої матриці ($X = A^{-1} \cdot B$).
    * Попереднє приведення системи до вигляду $AX - B = 0$.
    * Метод Гаусса (прямий та зворотний хід).

### **Частина B: Лінійне програмування (Стандартні задачі)** 
* Пошук оптимального розв'язку ЗЛП (максимізація/мінімізація).
* Використання модифікованих жорданових виключень (МЖВ) для ітераційного перерахунку симплекс-таблиць.

### **Частина C: ЗЛП зі змішаними обмеженнями**
* Робота з системами обмежень типу `=`, `<=`, `>=`.
* Реалізація алгоритму **видалення нуль-рядків** симплекс-таблиці.
* Пошук опорного та оптимального розв'язків для складних систем.

## 📜 Генерація протоколів

Програма підтримує формування детальних протоколів обчислень у реальному часі:
* Відображення вхідних даних.
* Покроковий вивід матриць після кожної ітерації (ЗЖВ/МЖВ).
* Опис вибору розв'язувальних елементів та обчислення фінальних коренів.
